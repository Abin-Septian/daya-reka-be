# README

## Getting Started

### Import Database
1. Import the database from the `db.sql` file provided.

### Install Dependencies
2. Install all dependencies using either npm or yarn:

```bash
npm install
# or
yarn install
```

### Start the Application

3. To start the application, run the following command:

```bash
npm run start
# or
yarn start
```
